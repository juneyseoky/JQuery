[ 제이쿼리 .animate( ); 메서드 용법 ]


1. animate( ) 메서드 기본 형식 ➜ .animate(A, B, C);

  1) A ➜ CSS
  2) B ➜ Duration(듀레이션, 적용시간), 단위. 밀리세컨즈 
  3) C ➜ Call Back(콜백함수)는 애니메이션 종료 후
           즉시 실행되는 기능으로 익명함수로 구현합니다.

  • 참고. 콜백함수 형식 ➜  function( ) {   ...   }     


2. animate( ) 메서드 사용법

    $("선택자").animate(
        {"css속성": "css속성값"}, 
        적용시간, 
        function( ) { 
            $("선택자").메서드(내용);
        }     
    );
